Master's Thesis

As part of my master's thesis work, I spent almost 8 months (from the second fall of Febraury 2017 till the end of September 2017) at the Institute of Networked and Embedded Systems (NES) in Klagenfurt, Austria, working as a research assistant in field of mobile robots: in particular, the topic of my work is coordinated multi-robot exploration with energy constraints: the robots have to explore an unknown environment in a cooperative way to efficiently execute this task (e.g.: to reduce the time required to completely explore the unknown environment), but they have a limited battery life and therefore they have to occasionally recharge at some charging locations; however, since these locations are limited in number and they can accomodate only one robot at a time, the robots have to coordinate also their access to them.

More in details, the first part of my work requires to implement, using ROS and C++, some theoretical models already developed by Micha Rappaport and Christian Bettstetter for the scenarios described above. They also collected some data using a Python implementation of these models but instead of using ROS, which is known to be a very realistic framework, they used a simplified one; therefore, the second part of the work is the collection of data using the Stage simulator and the code written using ROS and then the analisys and comparison of these data with the one originally produced by the two researchers.

The publicly available version of the code can be found on this repository, while the remaining part of the work and the most updated and refactored version is on a private repository (since I'm not the only author but instead I extended already existing code, I couldn't make everything publicly available).
