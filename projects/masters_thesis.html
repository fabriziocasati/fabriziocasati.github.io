    Master Thesis

    As part of my Master Thesis work, I spent almost 8 months (from the second fall of Febraury 2017 till the end of September 2017) at the Institute of Networked and Embedded Systems (NES) in Klagenfurt (Austria) working as a research assistant as part of my master's thesis project.

    The topic of my thesis is coordinated multi-robot exploration with energy constraints: the robots have to explore an unknown environment in a cooperative way to efficiently execute this task (e.g.: to reduce the time required to complete it), but they have a limited battery life, so, to complete the explorarion, they have to occasionally recharge at some charging locations, but since these locations are limited in number and can accomodate only one robot per time, the robots have to coordinate also their access to them.

    More in details, the first part of my work is to implement in ROS, using C++, some theoretical models already developed by Micha Rappaport and Christian Bettstetter for the scenarios described above. They also collected some data using a Python implementation of these models but not using the ROS framework: therefore, the second part of the work is the collection of data using Stage with the code written using ROS and then the analisys and comparison of these data with the one originally produced by the two researchers.

    The publicly available version of the code is on GitHub: you can find it here (notice that it is not on a "secondary" branch, since it's still a work in progress).
