    EFTAIOS (Bachelor's Thesis)

    Co-author: Valerio Castelli

    For our bachelor's thesis, Valerio and I had to implement a board game called Escare From the Aliens in Outer Space (EFTAIOS) using Java. In particular, we had to write the server-side and client-side software to allow remote multiplayer games: the server implements the game logic and the server-side networking functionalites, while the client provides a graphical interface and a command line interface to play a game (the user can choose which one he/she prefers), plus the remaining networking part.

    Both the server and client programs are fully documented using Javadoc, and the game logic part of the server was also tested using JUnit. Moreover, we used SonarQube to check the quality of the code.
