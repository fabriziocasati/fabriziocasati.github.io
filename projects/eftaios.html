    EFTAIOS (Bachelor Thesis)

    Co-author: Valerio Castelli

    For the final year project of my bachelor's degree, we were to implement a board game called Escare From The Aliens In Outer Space (EFTAIOS) using Java: in particular, we had to implement the server and the client to allow the players to play multiplayer games remotely: the server implements the game logic, while the client provides a graphical interface or, if the user prefers, a command line interface to play a game.

    The code of both server and client are fully documented using Javadocs, and for the game logic there are also test cases. Moreover, we used SonarQube to check the quality of the code.

    The code in not publicly available, but you can download the server and the client programs.
